Desacoplando URLs

  • Uma prática recomendada é relacionar as URLs com o app, e não com o projeto.
  • Dessa forma, no diretório do app estarão definidos as URLs relativas, e não absolutas.
  • Assim, diferentes projetos podem usar um mesmo app com o mesmo padrão de urls.

Templates

  • Template é um arquivo que contém parâmetros de substituição
  • Valores são atribuidos a esses parâmetros em views.py
  • Templates permitem a separação entre a lógica e exibição. Designers criam as páginas e programadores as preenchem com conteúdo dinâmico.
  • Templates possuem tags e filters, mas não é lugar de se programar
  • Templates devem ficar em um diretório listado na variável TEMPLATE_DIRS

Template Filters

  • Alteram o valor de variáveis antes de serem exibidas
  • Úteis para manipularmos o mesmo conjunto de dados de formas diferentes no mesmo template
  • Syntaxe: variável | filtro1 | filtro2:argumentos
  • Exemplos:
    • Aluno: {{ aluno.nome|lower|capfirst }}
    • {{ linguagem.list|dictsort:"nome" }}
    • {{ bio|truncatewords:"30" }}
    • {{ pub_date|date:"F j, Y" }}

22/33

views.py

  • É um arquivo normal em python, que contém a parte lógica (funções) das páginas
  • Essas funções recebem alguns argumentos:
    • HttpRequests
    • Parâmetros das URLs
  • E retornam objetos HttpResponse ou alguma subclasse do HttpResponse:
    • HttpResponse: resposta normal. O argumento é o conteúdo a ser mostrado para o usuário
    • HttpResponseRedirect: Um redirect (302). O argumento é a URL para onde será redirecionado
    • HttpResponseForbidden: Retorna a página erro 403
    • HttpResponseServerError: Retorna a página erro 500
    • HttpResponseNotFound: Retorna a página erro 404

Forms

  • O Django possui uma API para facilitar a criação e manejo de formulários
  • Tem sintaxe semelhante à do models.py
  • Validação automática
  • Exemplo, formulário de contato (forms.py):
# -*-*- encoding: utf-8 -*-*-
from django import forms


class ContactForm(forms.Form):
    subject = forms.CharField(max_length=100)
    message = forms.CharField(widget=forms.Textarea)
    sender = forms.EmailField(required=False, label='Your e-mail address')
    cc_myself = forms.BooleanField(required=False)

    def clean_message(self):
        message = self.cleaned_data['message']
        num_words = len(message.split())
        if num_words < 4:
            raise forms.ValidationError("Not enough words!")
        return message

27/33

Forms

  • views.py:
from django.core.mail import send_mail
from django.http import HttpResponseRedirect
from django.shortcuts import render
from contato.forms import ContactForm


def contact(request):
    if request.method == 'POST':
        form = ContactForm(request.POST)

        if form.is_valid():
            subject = form.cleaned_data['subject']
            message = form.cleaned_data['message']
            sender = form.cleaned_data['sender']
            cc_myself = form.cleaned_data['cc_myself']

            recipients = ['julio@juliobs.com']
            if cc_myself:
                recipients.append(sender)

            send_mail(subject, message, sender, recipients)
            return HttpResponseRedirect('/contato/ok/')
    else:
        form = ContactForm()

    return render(request, 'contato.html', {'form': form})
