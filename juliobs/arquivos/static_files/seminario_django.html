Template Filters

  • Alteram o valor de variveis antes de serem exibidas
  • teis para manipularmos o mesmo conjunto de dados de formas diferentes no mesmo template
  • Syntaxe: varivel | filtro1 | filtro2:argumentos
  • Exemplos:
    • Aluno: {{ aluno.nome|lower|capfirst }}
    • {{ linguagem.list|dictsort:"nome" }}
    • {{ bio|truncatewords:"30" }}
    • {{ pub_date|date:"F j, Y" }}

views.py

  • um arquivo normal em python, que contm a parte lgica (funes) das pginas
  • Essas funes recebem alguns argumentos:
    • HttpRequests
    • Parmetros das URLs
  • E retornam objetos HttpResponse ou alguma subclasse do HttpResponse:
    • HttpResponse: resposta normal. O argumento o contedo a ser mostrado para o usurio
    • HttpResponseRedirect: Um redirect (302). O argumento a URL para onde ser redirecionado
    • HttpResponseForbidden: Retorna a pgina erro 403
    • HttpResponseServerError: Retorna a pgina erro 500
    • HttpResponseNotFound: Retorna a pgina erro 404

Forms

  • O Django possui uma API para facilitar a criao e manejo de formulrios
  • Tem sintaxe semelhante do models.py
  • Validao automtica
  • Exemplo, formulrio de contato (forms.py):
# -*-*- encoding: utf-8 -*-*-
from django import forms


class ContactForm(forms.Form):
    subject = forms.CharField(max_length=100)
    message = forms.CharField(widget=forms.Textarea)
    sender = forms.EmailField(required=False, label='Your e-mail address')
    cc_myself = forms.BooleanField(required=False)

    def clean_message(self):
        message = self.cleaned_data['message']
        num_words = len(message.split())
        if num_words < 4:
            raise forms.ValidationError("Not enough words!")
        return message

27/33

Forms

  • views.py:
from django.core.mail import send_mail
from django.http import HttpResponseRedirect
from django.shortcuts import render
from contato.forms import ContactForm


def contact(request):
    if request.method == 'POST':
        form = ContactForm(request.POST)

        if form.is_valid():
            subject = form.cleaned_data['subject']
            message = form.cleaned_data['message']
            sender = form.cleaned_data['sender']
            cc_myself = form.cleaned_data['cc_myself']

            recipients = ['julio@juliobs.com']
            if cc_myself:
                recipients.append(sender)

            send_mail(subject, message, sender, recipients)
            return HttpResponseRedirect('/contato/ok/')
    else:
        form = ContactForm()

    return render(request, 'contato.html', {'form': form})
